# Student Exam Score Predictor

This Machine Learning project predicts student exam scores using Linear Regression.

## Skills Used
- Data Cleaning
- Missing Value Handling
- Feature Engineering
- One-Hot Encoding
- Correlation Analysis
- Linear Regression
- Model Evaluation

## Model Performance
- R² Score: 0.73
- RMSE: 2.03
- MSE: 4.15

## Libraries Used
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Dataset
Student Performance Factors Dataset (Kaggle)
